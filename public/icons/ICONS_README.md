# Extension Icons

The extension requires icons in three sizes: 16x16, 48x48, and 128x128 pixels.

## Required Files:
- `icon-16.png` - 16x16 pixels (toolbar icon)
- `icon-48.png` - 48x48 pixels (extension management)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Design Guidelines:

The icon should represent:
- Translation/language (globe, speech bubble, or language symbols)
- Manga/comics (book, comic style)
- A combination of both

Suggested color schemes:
- Primary: Blue or purple (technology/translation)
- Secondary: Accent color for visibility

## Creating Icons:

You can create icons using:
1. **Design tools:** Figma, Adobe Illustrator, Inkscape
2. **Icon generators:** realfavicongenerator.net
3. **AI tools:** Midjourney, DALL-E, etc.

## Quick Placeholder:

For development, you can create simple placeholders with any image editor, or use a service like https://favicon.io/

## Format:

- File format: PNG
- Background: Transparent preferred (or solid color)
- Resolution: Exact sizes required (16x16, 48x48, 128x128)
